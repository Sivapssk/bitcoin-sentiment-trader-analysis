# bitcoin-sentiment-trader-analysis
Analyzing relationship between Bitcoin market sentiment (Fear/Greed Index) and trader performance using Hyperliquid historical data.

# 📊 Bitcoin Sentiment vs Trader Performance Analysis

## 📌 Objective

Analyze how market sentiment (Fear/Greed Index) impacts trader behavior and profitability.

---

## 📂 Datasets Used

* Bitcoin Fear & Greed Index
* Hyperliquid Historical Trading Data

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn

---

## 📈 Key Analysis

* Profitability vs Sentiment
* Trade behavior (Buy/Sell)
* Trader-level performance
* Correlation between sentiment score & PnL

---

## 🔍 Key Insights

* Traders perform better during **Greed phases**
* High volatility during **Extreme Fear**
* Certain traders outperform consistently

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
jupyter notebook
```
